# Endless Sea (temporary name) - Game Design Document

## Quick Overview

**Endless Sea** is the name of a game jam to which i'm participating, the official name of the project has been decided but is being kept secret.

The goal of the game is to create a disturbing post-apo sea ambiance through mundane activities portrayed as minigames.
